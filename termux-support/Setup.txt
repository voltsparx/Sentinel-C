Sentinel-C Termux Setup

This folder is for building Sentinel-C directly in Termux on Android.

Method 1: Script build (recommended)

1. Open Termux.
2. Go to the project root folder.
3. Run:
   bash termux-support/build-termux.sh
4. If required tools are missing, the script will ask for permission to install them.
5. After success, the binary is available at:
   build-termux/bin/sentinel-c
6. A release copy is also placed at:
   bin-releases/termux/releases/bin/sentinel-c

Useful options:
  --clean       remove old build directory first
  --jobs <n>    set parallel build jobs
  --yes         auto-approve missing package installation
  --no-install  do not install missing tools automatically

Method 2: Manual build

1. Install required tools:
   pkg update -y
   pkg install -y cmake clang make
2. Go to project root.
3. Configure:
   cmake -S . -B build-termux -DCMAKE_BUILD_TYPE=Release
4. Build:
   cmake --build build-termux --target sentinel-c --parallel
5. Run:
   ./build-termux/bin/sentinel-c --help
